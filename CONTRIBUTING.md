# Contribution Guide

## Commit Convention

| Commit Types	| Description   |
|---|---|
| `docs` | Adds documentation to the project. |     
| `code` | Adds code to the project. | 
| `fix`  | Modification in the structure. |
| `()`   | Optional. |

## Examples

```
"docs - (Addition of/in {file-name}, ) what was added"
```

```
"code - (Addition of/in {file-name}, ) what was added"
```

```
"fix - (Modification of/in {file-name}, ) what was modified"
```

# Guia de Contribuição

## Convenção de commits

|  Tipos de Commit | Descrição  |      
|---|---|
| `docs` | Adiciona documentação no projeto. |     
| `code` | Adiciona código no projeto. | 
| `fix`  | Modificação na estrutura. |
| `()`   | Opicional. |

## Exemplos

```
"docs - (Adição do/no {nome-do-arquivo}, ) o que foi adicionado"
```

```
"code - (Adição do/no {nome-do-arquivo}, ) o que foi adicionado"
```

```
"fix - (modificação do/no {nome-do-arquivo}, ) o que foi modificado"
```
